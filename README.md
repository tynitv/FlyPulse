# FlyPulse (Timed Survival Flight Engine) ✈️

**FlyPulse** is a temporary timed survival flight manager for **Paper 1.21.x** servers.

---

## Commands & Permissions

- /tempfly - Activates temporary survival flight (Permission: flypulse.use, Aliases: /tfly, /flytime)

---

## Keywords & Search Terms
